Pré requisitos:
- Python 3.5 ou posterior;
- Gerenciador de pacotes do Python (pip);
- hbase-1.2.1 ou posterior;
- couchdb 1.6.0 ou posterior.

Para instalar o gerenciador de pacotes no Ubuntu:
sudo apt install python3-pip

Para instalar as dependências do script:
pip3 install -r requirements.txt --user

-- Creating a table with a single column family called 'series'
create 'dicom', 'series'

--final structure:
    rowkey, {'series:seriesInstanceUid': dicom_dataset.SeriesInstanceUID,
             'series:sopInstanceUid': dicom_dataset.SOPInstanceUID,
             'series:content': dicom_or_binary_content})

retrieve example:

    get 'dicom', '1.3.6.1.4.1.14519.5.2.1.5099.8010.256809878238026661650178294515'


Links úteis:
    https://dcm4che.atlassian.net/wiki/spaces/d2/pages/1835038/A+Very+Basic+DICOM+Introduction


Criar base "dicom" no couchdb:
curl -X PUT http://127.0.0.1:5984/dicom

Para executar o script:
python3 script.py 000000.dcm
