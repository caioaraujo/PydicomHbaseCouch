Pré requisitos:
- Python 3.5 ou posterior;
- Gerenciador de pacotes do Python (pip);
- hbase-1.2.1 ou posterior;
- couchdb 1.6.0 ou posterior.

Para instalar o gerenciador de pacotes no Ubuntu:
sudo apt install python3-pip

Para instalar as dependências do script:
pip3 install -r requirements.txt --user

Para executar o script:
python3 script.py "caminho/para/pasta/imagem/dicom"
